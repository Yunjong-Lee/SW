 ### 아나콘다-가상환경 backup/복원
 - 가상환경 export
   : "conda env export > 파일명. yaml" 입력  
 
    ```  
    $ conda env export > D:\env_backup\py35.yml  
    ```  
   - 여기서 이 명령어를 시행하는 Prompt는 백업할 env의 프롬프트임.  
 
- 백업파일을 이용하여 가상환경 생성

    ```  
    $ conda env create --name py35 --file D:\env_backup\py35.yml  
    ```
